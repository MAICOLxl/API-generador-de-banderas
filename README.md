# API-generador-de-banderas

Es una herramienta web que les permita consultar información general de distintos países y compararlos a partir de variables demográficas 
y geográficas, como población, capital, región, bandera e idioma.

La herramienta te permite compararar de 2 a 5 paises al mismo tiempo generando una tabla de comparacion con las varibles mencionadas.

Cuenta con la intregracion de la siguiente api https://restcountries.com/ para buscar la informacion actulizada de los paises.
